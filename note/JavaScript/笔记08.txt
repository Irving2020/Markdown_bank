    var a = b = 10;

递归
    - 函数自己调用自己就是递归
    - 递归的核心思想就是将一个大的问题拆分为一个一个小问题
    - 递归的作用和循环的作用是类似，不同点在于：
          循环执行的速度比较快，代码不太好理解
          递归的执行速度慢，代码比较容易理解

Date
    - Date是JS中的日期对象
    - JS中所有的时间相关的内容都使用Date来表示
    - 创建Date对象：
        1.创建一个表示当前日期的Date
            let d = new Date();
        2.创建一个指定日期的Date
            let d = new Date('月/日/四位年 时:分:秒');
    - 对象中的方法：
        Date.now() （类方法、静态方法）
            - 获取当前的时间戳
        d.getTime()
            - 获取当前日期对象的时间戳
        d.getDay()
            - 获取当前日期是周几
            - 返回值 0-6
                0周日，1周一 ...

        d.getMonth()
            - 获取当前日期是几月
            - 返回值：0-11
                0一月，1二月...

        d.getDate()
            - 获取当前日期是几日

        d.getFullYear()
            - 获取当前日期对象的年份

包装类
    - JS中一共提供了三个包装类
        String() Number() Boolean()
    - 包装类可以用来创建基本数据类型的对象
        包装类是留给浏览器自己用的，
        当我们调用一共基本数据类型的属性（方法）时，
            浏览器会临时将其转换为对象，然后调用对象的属性（方法）
            调用完成后，对即被销毁

字符串
    - 在底层字符串就是以字符数组形式存在的
    - 'abc' --> ['a', 'b', 'c']
    - 字符串的方法：
        str[索引]
            - 根据索引获取字符
        str.length
            - 获取字符串的长度
        str.charAt()
            - 根据索引获取字符
        str.charCodeAt()
            - 根据索引获取字符编码
        String.fromCharCode()
            - 根据编码返回字符
        str.indexOf()
            - 查询子串在字符串中第一次出现的位置
        str.lastIndexOf()
            - 查询子串在字符串中最后一次出现的位置
        str.slice()
            - 截取字符串
            - 参数：
                1. 截取的起始位置（包含）
                2. 截取的结束位置（不包含）
        str.substr()
            - 截取字符串
            - 参数：
                1. 截取的起始位置
                2. 截取的长度
        str.toUpperCase()
            - 转换为大写
        str.toLowerCase()
            - 转换为小写
        str.trim()
            - 去除字符串中的空格
        str.split()
            - 将字符串拆分为一个数组
            - 参数：
                字符串：分隔符
        str.concat()
            - 用来将两个或两个以上的字符串拼接为一个字符串

        ---------------------------------------------

        str.trimEnd()
            - 去除后边的空格
        str.trimStart()
            - 去除开头的空格
        str.startsWith()
            - 检查字符串是否以指定内容开头
        str.endsWith()
            - 检查字符串是否以指定内容结尾
